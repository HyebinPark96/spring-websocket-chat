# spring-websocket-chat
📌 Spring Boot + WebSocket 채팅 프로그램 구현  
* Java Version 8, Gradle / Spring Boot 
* 사용한 라이브러리 
  stomp-websocket & sockjs 
## 기능
  * 1:N 통신
  * 일반 채팅 및 귓속말 
    * /w {id} 입력 시 귓속말 모드
  * 사용자 관리 (구현중)
    * 사용자 정보 확인
    * 강제 연결 종료
  * 채팅방 생성
    * 채팅방 이름 및 인원 수 제한
  * 방장의 강제 퇴장 권한 및 역임 제도
